Mario Kart 8 Trainer v2.0 by Bully@WiiPlaza

This application can connect to your Nintendo Wii U console over the local network and perform memory modifications. In order for this to work, you need to run the Homebrew Launcher and my TCP Gecko Installer before starting Mario Kart 8. Only game version 4.1 is supported since it's the latest available so make sure you update yours.

Make sure that your JGecko U or any other client application is disconnected when trying to apply any patches since they will not take effect until then.

For bug reports or comments on this application, can contact me on YouTube, GBATemp or BullyWiiHacks.
