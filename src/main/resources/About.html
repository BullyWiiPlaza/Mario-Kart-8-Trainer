Mario Kart 8 Trainer v1.3 by Bully@WiiPlaza

This application can connect to your Nintendo Wii U console over the local network and perform memory modifications. In order for this to work, you need to run the kernel exploit and PyGecko installer before starting Mario Kart 8. Only Mario Kart 8 version 4.1 is supported since it's the latest available so make sure you update yours. The Wii U natively has to be on firmware 5.3.2 and below to be publicly exploitable as of now.

Make sure that your TCP Gecko.NET or any other application that can connect to the Wii U is disconnected when trying to apply any patches since they will not take effect until then.
